## Any Launcher - 轻量多功能游戏启动器

Any Launcher 是一款基于 Python 开发的轻量级游戏启动器，支持游戏导入、管理、快速启动等核心功能，适配 exe 与 HTML 格式游戏，具备配置自动保存、上次选中游戏记忆、文件完整性校验等实用特性，同时优化了中文路径兼容、权限适配等常见问题，提供流畅的游戏管理体验。

### ✨ 核心功能

- 游戏导入：支持选择本地 exe/HTML 游戏文件，自动提取默认名称并生成文件 MD5 校验码
- 游戏管理：支持游戏名称/路径修改、单游戏精准删除（基于唯一 ID 避免误删）、游戏列表刷新
- 快速启动：记忆上次选中游戏，一键启动，支持启动后最小化启动器
- 文件校验：导入游戏时自动计算 MD5 值，启动前校验文件完整性，防止文件被篡改或损坏
- 兼容性优化：完美支持中文/特殊字符路径，默认将配置文件存储在 AppData 目录（避免系统目录权限问题）
- 友好交互：清晰的错误提示、操作确认弹窗，支持窗口层级管理，避免多窗口干扰

### 📋 环境要求

- Python 版本：3.8 及以上
- 依赖库：customtkinter、pillow、filelock
- 系统支持：Windows 7/10/11（32/64 位）

### 🚀 安装步骤

#### 1. 依赖安装

克隆项目后，在项目根目录执行以下命令安装依赖：

```bash
pip install customtkinter pillow filelock
```

### 📖 使用指南

#### 1. 导入游戏

1. 运行程序，点击「选择游戏」按钮打开游戏选择窗口
2. 点击「导入游戏」，在文件选择窗口中选中本地游戏的 exe 或 HTML 文件
3. 在弹出的窗口中输入游戏名称（默认提取文件名，可修改），点击「确定」完成导入

#### 2. 管理游戏

- 修改游戏：在游戏选择窗口选中游戏，点击「游戏设置」，可修改名称、重新选择游戏路径（自动更新 MD5）
- 删除游戏：在游戏选择窗口选中游戏，点击「删除游戏」，确认后即可删除（仅删除选中游戏，不影响本地游戏文件）
- 刷新列表：若手动修改了配置文件，点击「刷新列表」可重新加载游戏数据

#### 3. 启动游戏

1. 通过「选择游戏」窗口选中目标游戏，点击「选择并关闭」返回主窗口
2. 主窗口会显示当前选中的游戏名称，点击「启动游戏」即可启动
3. 启动成功后，可选择是否最小化启动器（不影响游戏运行）

### 🔧 常见问题解决

- Q：导入游戏时提示「无写入权限」？ A：请以管理员身份运行程序，或手动将配置文件路径（AppData/Roaming/Any Launcher）设置为可写权限。
- Q：启动游戏时提示「文件不存在」但文件实际存在？ A：可能是路径包含特殊字符，建议通过「游戏设置」重新选择游戏路径，程序会自动处理路径编码。
- Q：配置文件损坏无法加载？ A：删除 AppData/Roaming/Any Launcher 目录下的 games.json 文件，重新导入游戏即可。
- Q：Logo 显示异常？ A：检查项目根目录是否存在 logo.png 文件，或替换为符合尺寸的图片（建议尺寸 400x200）。
